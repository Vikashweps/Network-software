# Интернет-технологии и сетевое ПО

Добро пожаловать в репозиторий с практическими заданиями. Здесь мы шаг за шагом разберём основные инструменты, которые используются в современной разработке: от Docker и Kubernetes до CI/CD пайплайнов сборки и доставки программных продуктов.

В каждой папке (`weeks/week-XX`) вас ждут:
- **README.md** — немного теории, чтобы погрузиться в тему, что нужно сделать и как запустить.
- **questions.md** — вопросы для самопроверки.
- **tests/** — автоматические тесты, которые помогут проверить решение.

Основной язык для выполнения заданий — **Python (FastAPI)**

## С чего начать

1. **Подготовим окружение**
   Убедитесь, что у вас установлен Python (версии 3.11 или выше).
   ```bash
   python -m venv .venv && source .venv/bin/activate
   pip install -r requirements.txt
   ```

2. **Найдите себя в списке**
   В файле `students/roster.csv` найдите свою группу и ID.
   Экспортируйте их в переменные окружения, чтобы тесты знали, какой вариант проверять:
   ```bash
   export GROUP=331
   export STUDENT_ID=s16
   ```

3. **Запустите тесты**
   Для проверки заданий первой недели используйте команду:
   ```bash
   make test WEEK=01
   ```

## Ваши индивидуальные задания
Для каждого сгенерирован свой вариант задания. Он лежит в JSON-файле:
`variants/<GROUP>/<STUDENT_ID>/week-XX.json`.

Вы можете быстро посмотреть свой вариант командой:
```bash
python tools/show_my_variant.py 02
```

Тесты автоматически считывают этот файл, используя ваши переменные окружения, и проверяют, соответствует ли ваше решение заданию.
